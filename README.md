# docsify

## 快速开始

* [官方中文文档](https://docsify.js.org/#/zh-cn/quickstart)

* 安装 docsify-cli

```shell
docsify-cli
```

* 初始化项目

```shell
docsify init ./docs
```

## 配置

* [codefund](https://codefund.io/)

## docs

* 本地预览网站

```shell
docsify serve docs

# 或

docsify serve ./docs
```
